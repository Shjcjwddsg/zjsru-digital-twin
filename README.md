# 浙江树人学院（拱宸桥校区）数字孪生交互系统

## 📋 项目简介

本项目是一个基于 **Three.js** 技术构建的**浙江树人学院拱宸桥校区数字孪生交互系统**。该系统通过Web 3D可视化技术，将真实的校园环境以三维形式呈现在浏览器中，用户可以通过鼠标交互浏览校园、查看建筑信息、切换不同视角和图层。

**校园特点：**
- 分为**南校区**和**北校区**
- 中间由**舟山东路**隔开
- 包含完整的**校园围墙**和校门
- 从**南校区南门**为起始视角

> **⚠️ 重要说明：本项目是浙江树人学院期中作业**

## ✨ 核心功能特性

### 🏛️ 三维场景建模
- **11栋主要建筑**：图书馆、教学楼（2栋）、实验楼、行政楼、体育馆、学生活动中心、食堂、宿舍楼（2栋）、信息中心
- **道路系统**：主干道、支路及道路标线
- **绿化植被**：树木、灌木等自然景观
- **体育设施**：标准足球场（含跑道）、篮球场

### 📷 多视角切换（4种视角）
1. **透视视图** - 默认的3D透视效果，适合整体浏览
2. **鸟瞰视图** - 从正上方俯瞰整个校园布局
3. **正视图** - 从正面观察校园建筑群
4. **侧视图** - 从侧面查看校园空间结构

### 🖱️ 交互控制系统（3类以上）
- **鼠标拖拽旋转** - 360°自由旋转查看场景
- **滚轮缩放** - 放大/缩小场景细节
- **右键平移** - 移动视角位置
- **点击选择** - 点击建筑物显示详细信息

### 🗂️ 图层管理功能（4个图层）
- **建筑物图层** - 独立显示/隐藏所有建筑
- **道路系统图层** - 控制道路可见性
- **绿化植被图层** - 控制树木灌木显示
- **体育设施图层** - 控制运动场地显示

### 💡 核心功能模块
1. **图文介绍面板** - 展示校园概况和基本信息
2. **对象信息展示** - 点击任意建筑查看详细信息（名称、层数、面积、建成时间）
3. **视角变换系统** - 平滑的相机动画过渡效果
4. **日夜模式切换** - 模拟白天/夜晚场景，夜间建筑灯光闪烁效果
5. **自动旋转演示** - 全景自动展示功能
6. **重置视角** - 一键恢复默认视角

## 🛠️ 技术栈

- **前端框架**: 原生 JavaScript (ES6+)
- **3D引擎**: Three.js (v0.160.0)
- **控制器**: OrbitControls
- **样式**: CSS3 (Flexbox, Grid, 动画)
- **渲染**: WebGL (硬件加速)

## 🚀 快速开始

### 方式一：直接打开（推荐）
1. 克隆或下载本项目到本地
2. 双击 `index.html` 文件即可在浏览器中运行
3. （部分浏览器可能需要本地服务器）

### 方式二：使用本地服务器
```bash
# 安装依赖（可选，用于开发服务器）
npm install

# 启动开发服务器
npm start
# 或
npm run dev

# 访问 http://localhost:8080
```

### 方式三：VS Code Live Server
1. 使用 VS Code 打开项目
2. 安装 "Live Server" 扩展
3. 右键 `index.html` → "Open with Live Server"

## 🌟 高级特效

### 1. 日夜循环系统
- 白天模式：明亮的天空、自然光照
- 夜晚模式：深蓝色天空、建筑窗户发光、灯光闪烁动画

### 2. 平滑相机过渡
- 所有视角切换都采用缓动函数（easeInOutCubic）
- 流畅的动画体验

### 3. 实时阴影
- 所有物体投射实时阴影
- 增强场景真实感

### 4. 雾化效果
- 距离雾增强空间深度感
- 日夜模式下雾气颜色自适应
